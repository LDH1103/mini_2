Controller는 Model과 View 간의 중개자 역할을 수행
사용자의 입력을 받아 View에서 이벤트가 발생하면, 해당 이벤트를 분석하고,
이에 따라 Model을 업데이트하거나, View를 업데이트함

**
파일 반환: return "main"._EXTENSION_PHP;

파일 반환은 서버가 클라이언트에게 특정 파일의 내용을 반환하는 동작
서버는 해당 파일의 내용을 가져와서 응답으로 전송하고, 클라이언트는 받은 파일을 처리하여 표시하게 됨
주로 웹 애플리케이션에서 특정 페이지의 내용이나 리소스 파일(HTML, CSS, JavaScript, img 등)을 클라이언트에게 제공하는데 사용
파일 반환은 해당 파일이 서버에 존재하고 그 내용을 클라이언트에게 보여주는 것에 초점을 둠


주소 리디렉션: return _BASE_REDIRECT."/shop/main";

주소 리디렉션은 클라이언트를 다른 URL로 이동시키는 동작
서버는 특정 주소로 리디렉션하는 응답을 클라이언트에게 보내고, 클라이언트는 받은 리디렉션 응답에 따라 새로운 URL로 이동하게됨
주소 리디렉션은 클라이언트를 다른 페이지나 경로로 이동시키는데 사용
주소 리디렉션은 해당 페이지가 존재하지 않거나 사용자를 다른 관련 페이지로 안내하거나, 
다른 도메인 또는 서브도메인으로 이동시키는 등의 기능을 구현할 때 사용
